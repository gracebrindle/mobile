Visualizing the Motor City

As an Engagement Fellow at the University of Michigan Library, I am currently working with a film librarian to build a database of theaters and films in Detroit. This database will be used in an undergraduate film course next semester where students will use and build upon the data in their own digital projects.

The first stage of this process involved creating a data dictionary and defining our metadata fields. We examined the data that had already been collected, decided on a set of metadata fields, and standardized the format of these fields. This process ensured that our data entry remained consistent and that our database would be easy for users to understand.

The second and current stage of this process entails collecting the data. I am using a web scraping tool to collect the data from public websites and databases. Once this phase is complete, I will input the data into ArcGIS and begin to create some interesting visualizations.
